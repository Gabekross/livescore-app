# Match Management

## Match Types

### Tournament Matches
- Belong to a tournament group
- Affect standings
- Scheduled within a group's match list

### Friendly Matches
- Standalone games with no tournament association
- Never affect standings
- Labeled as "Friendly" on the public site
- Created from the dashboard via **Friendly Match**

## Scheduling a Match

### Tournament Match
1. Navigate to a tournament → stage → group
2. Click **New Match**
3. Select home and away teams (from teams assigned to the group)
4. Set the match date and time
5. Save — the match appears as "Scheduled"

### Friendly Match
1. From the dashboard, click **Friendly Match**
2. Select any two teams from your organization
3. Set the date and time
4. Save

## Live Match Updates

When it's match day:

1. Open the match (from the match list or Operator View)
2. Change the match status to **Live**
3. Update the score as goals are scored
4. Record events: goals, cards, substitutions
5. When the match ends, set status to **Completed**

Changes appear on the public site instantly.

## Match Operators (Pro)

Match Operators are users with limited access who can update live scores:
- They can only see and update assigned matches
- They cannot access other admin features
- Ideal for delegating match-day score updates

Set up operators from **Match Operators** in the dashboard.

## Operator View

The Operator View (`/admin/operator`) is a streamlined panel for live match updates:
- Shows only matches that need attention
- Simplified interface for fast score entry
- Works well on mobile devices for pitch-side updates

## Formation Editor

For matches with lineups:
1. Open a match
2. Access the **Formation Editor**
3. Choose a formation preset (e.g., 4-3-3, 4-4-2)
4. Drag players to position them on the pitch
5. Save the lineup

Formations display on the public match detail page.
