# News & Media (Pro)

## Publishing News Articles

News publishing is a Pro feature. To create an article:

1. From the dashboard, click **News & Articles**
2. Click **New Post**
3. Write your article using the rich text editor:
   - Add headings, bold, italic text
   - Insert links
   - Embed images from your Media Library
4. Add a **featured image** for the article card
5. Choose a **category** for organization
6. Click **Publish**

### Article Editing
- Edit any published article from the News list
- Changes take effect immediately on your public site
- You can unpublish articles to hide them from visitors

## Media Library

The Media Library is your central hub for images and videos:

1. From the dashboard, click **Media Library**
2. Upload images by dragging and dropping or using the file picker
3. Supported formats: **JPG, PNG, WebP**
4. Use uploaded media in news articles, as team logos, or tournament covers

### Best Practices
- Use high-quality images (at least 1200px wide for featured images)
- Keep file sizes reasonable for fast loading
- Use descriptive filenames for better organization

## Using Media in Articles

The rich text editor includes a **Media Picker** that lets you:
1. Browse your uploaded media
2. Select an image to insert into the article
3. The image is automatically optimized for web display

## Tips

- Featured images make articles stand out in the news feed
- Consistent image sizing creates a more professional look
- Write compelling headlines — they appear on the homepage and social shares
